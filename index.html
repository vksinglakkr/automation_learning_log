About this log

Why this exists

The Automation Log documents a self-taught n8n journey built for real district administration needs during a career at the National Informatics Centre, Govt. of India — starting from a single scheduled message and ending in a full event-support system.

The last build, the Hackathon Management System, was a practice exercise — a virtual scenario used to learn end-to-end event operations (registration correction, categorized support triage, FAQ handling, and event-day coupon management) without a live event behind it.

This page now serves as a reference for students being introduced to n8n from scratch — each project doubles as a lesson, complete with the real node chain and the concepts it was chosen to teach.
